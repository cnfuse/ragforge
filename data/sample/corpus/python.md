# Python

Python is a high-level, interpreted programming language created by Guido van
Rossum and first released in 1991. It emphasises code readability with
significant indentation and a clean, English-like syntax.

Python supports multiple paradigms — procedural, object-oriented, and
functional. Its extensive standard library and the PyPI ecosystem make it a
popular choice for web development, data science, automation, and machine
learning.
